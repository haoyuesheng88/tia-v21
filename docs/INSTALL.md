# Install

## Skill

Copy this folder:

`skills/tia-openness-closed-loop`

to:

`%USERPROFILE%\.codex\skills\`

## Examples and tools

The `examples` and `tools` folders can stay in any workspace.

## Prerequisites

- `TIA Portal V21`
- `TIA Portal Openness V21`
- A Windows account that is a member of:
  - `Siemens TIA Openness`
- A local C# compiler or Visual Studio Build Tools if you want to build the helper utilities

## Typical workflow

1. Install the skill.
2. Open the target TIA project.
3. Use the skill instructions in `SKILL.md`.
4. Build and run only the helper source files you need for that task.
